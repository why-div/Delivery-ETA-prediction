# Delivery ETA Prediction

## Problem
Predict delivery time for e-commerce orders using order, seller, and customer data, to help set accurate customer-facing ETAs.

## Dataset
Olist Brazilian E-commerce dataset (Kaggle) — orders, order items,sellers, customers, geolocation tables.

## Approach
1. SQL exploration of relational tables
2. EDA on delivery time distribution and drivers
3. Feature engineering (distance, seller history, order timing)
4. Baseline vs ML models (Linear Regression, XGBoost)
5. Error analysis by region/category
6. Deployment via Streamlit
